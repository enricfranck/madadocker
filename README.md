
Pour le backend

creer un environement virtuel
```bash
cd backend
python3 -m venv venv
```

activer l' environement virtuel
```bash
source venv/bin/activate
```
installer les packages
```bash
pip3 install -r requirements.txt
```
lancer le backend
```bash
python3 app/main.py 
```

http://localhost:8888/api/v1

http://localhost:8888/api/docs

Pour le frontend
lancer le backend
```bash
cd frontend
```
installer les packages
```bash
npm install
```
lancer le front
```bash
npm run start
```

http://localhost:4200
